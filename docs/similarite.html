Calcul de la similarité¶

similarity.similarite()¶

Cette fonction réalise l’enchaînement des fonctions permettant d’obtenir le coefficent de similarité d’un modèle de données

Paramètres:
  • model – modèle de données text8 récupéré à partir de wikipedia anglais
  • matrice – jeu de données analysé, rendu sous forme de matrice
Retourne:

Retourne le coefficent de similarité entre le modèle text8 et le jeu de données sous forme de matrice fourni en entrée
